# Flask Learning 🚀

This repository contains basic Flask practice projects and examples created while learning Flask for web development.

## 📌 About

This project demonstrates the fundamentals of building web applications using Flask in Python.
It includes basic routing, form handling, sessions, and authentication logic.

## 🛠️ Technologies Used

* Python
* Flask
* HTML (Basic Forms)

## 📂 Project Features

* Flask routing (`/`, `/about`, `/contact`)
* Login form using POST method
* Session management
* Redirect after login
* Logout functionality

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/ayushsingh01894/Learning_frameworks.git
```

2. Move into the project folder

```
cd flask_learning
```

3. Install Flask

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
